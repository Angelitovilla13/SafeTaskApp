SafeTaskApp

Aplicación web para gestión de tareas y proyectos personales, con inicio de sesión seguro y roles de usuario (administrador y usuario estándar).

Usuarios objetivo: estudiantes y profesionales.

2) Roles, permisos y gestión de tareas

Administrador: puede ver y gestionar todas las tareas.
Usuario estándar: solo puede gestionar sus propias tareas.

    3) Seguridad y buenas prácticas solicitadas

    • HTTPS obligatorio en producción.
    • Protección XSS con Content Security Policy.
    • Prevención CSRF con tokens en formularios (backend).
    • Prevención SQL Injection con ORM/consultas parametrizadas.
    • Rate limiting y bloqueo temporal anti fuerza bruta.

    4) Notificaciones

    Se muestran alertas internas para tareas próximas a vencer (48h).

      5) Tecnologías recomendadas

      • Frontend: HTML5, CSS3, JavaScript.
      • Backend: Node.js + Express / Python + Django / PHP + Laravel.
      • Base de datos: MySQL.
      • Autenticación: sesiones seguras.
      • Hosting: VPS o PaaS (Render, Railway, etc.).
